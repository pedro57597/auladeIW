Javascript é uma linguagem de programação de alta complexidade, mas de fácil uso, voltada para criar elementos em aplicações web como sites, aplicativos e sistemas online....

Entre as principais linguagens utilizadas em larga escala mundialmente está a JavaScript, principalmente no desenvolvimento para a web.

  1. HTML – Permite adicionar conteúdos em uma página através de marcação de textos. É a estrutura óssea de um site.
  2. CSS – Especifica o estilo do design do layout de uma página. É tudo relacionado à aparência de um site.
    3.  
  3. PHP – Dinamiza o processo de desenvolvimento de uma página adicionando extensões e aplicações dinâmicas.
  4. JavaScript – Controla o comportamento dos elementos de uma página, executando funções dinâmicos dos conteúdos, animações, aplicativos e interações em geral.
